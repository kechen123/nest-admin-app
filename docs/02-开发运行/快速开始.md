## 🎯 最常用的 5 个命令（新手专用）

如果你只想快速启动项目，记住这 5 个命令就够了：

1. **启动所有服务**
   ```bash
   npm run dev:up
   ```
   说明：启动数据库、后端、前端三个服务

2. **初始化数据库**（第一次运行时需要）
   ```bash
   npm run backend:init-db
   ```
   说明：创建数据库和初始数据

3. **查看服务状态**
   ```bash
   npm run ps
   ```
   说明：看看哪些服务在运行

4. **停止所有服务**
   ```bash
   npm run dev:down
   ```
   说明：关闭所有服务

5. **查看日志**（出问题时用）
   ```bash
   npm run dev:logs
   ```
   说明：查看所有服务的运行日志，按 Ctrl+C 退出

**就这么简单！** 其他命令都是高级功能，暂时不需要了解。

## 🚀 Docker 运行（推荐）

```bash
# 启动所有服务（后台运行）
npm run dev:up

# 等待 MySQL 启动后（约 20-30 秒），初始化数据库（首次运行）
npm run backend:init-db

# 查看服务状态
npm run ps
```

- **前端应用**: http://localhost:4000
- **后端 API**: http://localhost:3000/api

## 💻 本地运行

**适用场景**：后端在本地运行，MySQL 在 Docker 容器中

```bash
# 启动 MySQL 容器
npm run mysql:start

# 初始化数据库（在 backend 目录）
cd backend && npm run db:init

# 启动后端服务
npm run start:dev
```

详细说明请查看 [本地运行](./本地运行.md)
