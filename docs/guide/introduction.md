# 项目介绍

欢迎来到 yl 项目文档！

## 项目概述

yl 是一个基于 Docker 的现代化全栈开发解决方案，旨在提供一套完整的开发、部署和管理工具链。

## 项目特点

- 🐳 **容器化部署**：使用 Docker Compose 统一管理所有服务
- 🚀 **快速开发**：内置热重载和开发工具，提升开发效率
- 📦 **服务管理**：统一管理前端、后端、数据库等服务
- 🔧 **便捷部署**：提供自动化部署脚本
- 📊 **健康检查**：实时监控服务状态

## 技术栈

### 前端
- 现代化的前端框架
- 响应式设计
- 丰富的 UI 组件

### 后端
- RESTful API
- 数据库集成
- 完善的错误处理

### 基础设施
- Docker & Docker Compose
- MySQL 数据库
- 自动化部署工具

## 运行方式

项目支持两种运行方式，您可以根据需求选择：

- **[Docker 运行](./getting-started.md#docker-运行)** - 推荐方式，使用 Docker Compose 一键启动所有服务
- **[pnpm 本地运行](./getting-started.md#pnpm-本地运行)** - 适合需要本地开发调试的场景
