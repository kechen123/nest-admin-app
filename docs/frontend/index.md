## 开发环境

前端开发服务器支持热重载，修改代码后会自动刷新浏览器。

### 启动前端服务

```bash
npm run web:start
```

### 查看日志

```bash
npm run web:logs
```

### 进入容器

```bash
npm run web:shell
```
