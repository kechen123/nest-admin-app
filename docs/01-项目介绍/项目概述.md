# 项目概述

## 项目简介

本项目是一个全栈管理系统，包含三个子项目：

- **backend**: NestJS 后端服务
- **web**: Vue 3 + Element Plus 后台管理前端
- **uniapp-mobile**: UniApp 跨端移动应用

## 运行方式

项目支持 **Docker 运行** 和 **本地运行** 两种方式。

### Docker 运行（推荐）

所有服务（数据库、后端、前端）都在 Docker 容器中运行，最简单，推荐新手使用。

**优点**：
- 环境统一，无需手动配置
- 一键启动，简单快捷
- 支持热重载，开发体验好

### 本地运行

后端和前端在你的电脑上直接运行，只有数据库在 Docker 中，适合需要调试代码的情况。

**优点**：
- 可以直接调试代码
- 修改代码立即生效
- 适合深度开发

## 适用场景

- **新手学习**：推荐使用 Docker 模式，更简单，不容易出错
- **代码调试**：推荐使用本地模式，可以直接调试代码
- **生产部署**：使用 Docker 构建生产镜像，部署到服务器
