# 技术栈

## 后端技术栈

### 核心框架

- **NestJS** - 渐进式 Node.js 框架，用于构建高效、可扩展的服务器端应用程序
- **TypeScript** - JavaScript 的超集，提供类型安全
- **Node.js** - JavaScript 运行时环境

### 数据库

- **MySQL 8.0** - 关系型数据库
- **TypeORM** - ORM 框架，用于数据库操作

### 认证与安全

- **JWT** - JSON Web Token，用于用户认证
- **bcrypt** - 密码加密

### 文档

- **Swagger** - API 文档生成工具

### 其他

- **class-validator** - 数据验证
- **class-transformer** - 数据转换

## 前端技术栈

### 核心框架

- **Vue 3.5+** - 渐进式 JavaScript 框架（使用 Composition API）
- **TypeScript 5.8+** - 类型安全的 JavaScript
- **Vite 6+** - 下一代前端构建工具

### UI 框架

- **Element Plus 2.10+** - 基于 Vue 3 的组件库

### 状态管理

- **Pinia 3.0+** - Vue 的状态管理库

### 路由

- **Vue Router 4.5+** - Vue.js 官方路由管理器（自动路由）

### HTTP 客户端

- **Axios 1.9+** - 基于 Promise 的 HTTP 客户端

### 图表库

- **ECharts 5.6+** - 数据可视化图表库

### 样式

- **Less 4.3+** - CSS 预处理器

### 代码质量

- **ESLint** - JavaScript 代码检查工具
- **Prettier** - 代码格式化工具

## 移动端技术栈

- **UniApp** - 跨平台应用开发框架
- **Vue 3** - 前端框架
- **TypeScript** - 类型安全

## 开发工具

### 容器化

- **Docker** - 容器化平台
- **Docker Compose** - 多容器应用编排工具

### 包管理

- **npm** - Node.js 包管理器
- **pnpm** - 快速、节省磁盘空间的包管理器（推荐）

### 版本控制

- **Git** - 分布式版本控制系统

## 环境要求

### 开发环境

- Node.js >= 18.0.0
- Docker >= 20.10.0
- Docker Compose >= 2.0.0
- npm >= 9.0.0 或 pnpm >= 8.0.0

### 生产环境

- Node.js >= 18.0.0
- Docker >= 20.10.0
- Docker Compose >= 2.0.0
- MySQL 8.0+
