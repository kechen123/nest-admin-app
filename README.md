# 全栈项目

本项目包含三个子项目：

- **backend**: NestJS 后端服务
- **admin-frontend**: Vue 3 + Element Plus 后台管理前端
- **uniapp-mobile**: UniApp 跨端移动应用

## 项目结构

```
my-fullstack-project/
├── backend/                 # NestJS 后端服务
├── admin-frontend/          # Vue3 + Element Plus 后台管理
├── uniapp-mobile/           # UniApp 移动端项目
├── docs/                    # 项目文档
└── README.md
```

## 快速开始

### 后端服务

```bash
cd backend
npm install
# 配置 .env 文件
npm run start:dev
```

### 后台管理前端

```bash
cd admin-frontend
npm install
npm run dev
```

### 移动端应用

```bash
cd uniapp-mobile
npm install
# 使用 HBuilderX 或 CLI 运行
```
