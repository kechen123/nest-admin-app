## ⚡ 快速开始

```bash
# 启动开发环境
npm run dev:up

# 初始化数据库
npm run backend:init-db
```

更多信息请查看 [快速开始指南](./docs/02-开发运行/快速开始.md)

## 🎯 最常用的 5 个命令

1. **启动所有服务**
   ```bash
   npm run dev:up
   ```

2. **初始化数据库**（第一次运行时需要）
   ```bash
   npm run backend:init-db
   ```

3. **查看服务状态**
   ```bash
   npm run ps
   ```

4. **停止所有服务**
   ```bash
   npm run dev:down
   ```

5. **查看日志**（出问题时用）
   ```bash
   npm run dev:logs
   ```

## 📝 项目简介

本项目包含三个子项目：

- **backend**: NestJS 后端服务
- **web**: Vue 3 + Element Plus 后台管理前端
- **uniapp-mobile**: UniApp 跨端移动应用

支持 **Docker 运行** 和 **本地运行** 两种方式。
