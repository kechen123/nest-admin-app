# 恋爱足迹打卡小程序

基于 uniapp 开发的恋爱足迹打卡小程序，支持在地图上标点、上传图片和文字打卡记录约会时光。

## 功能特性

### 1. 地图打卡
- 📍 在地图上显示所有打卡点
- 🗺️ 支持地图缩放和定位
- 📌 点击标记点查看打卡详情

### 2. 发布打卡
- 📍 选择位置（支持当前位置或手动选择）
- 📷 上传图片（最多9张）
- ✍️ 输入打卡文字内容
- 💾 保存打卡记录

### 3. 打卡记录
- 📝 按日期分组的打卡记录列表
- 🕐 时间轴展示
- 📸 图片预览
- 🔍 查看详情

### 4. 打卡详情
- 📍 位置信息展示
- 📷 图片轮播
- ✍️ 文字内容
- 🗑️ 删除打卡记录

### 5. 首页统计
- 📊 总打卡数、本月打卡、本周打卡统计
- 🎯 快捷操作入口
- 📰 最新打卡展示

## 页面结构

```
src/pages/
├── index/          # 首页（统计和快捷入口）
├── map/            # 地图页面（显示所有打卡点）
├── checkin/
│   ├── add.vue     # 发布打卡页面
│   ├── list.vue    # 打卡记录列表
│   └── detail.vue  # 打卡详情页面
└── me/             # 个人中心
```

## 数据存储

使用 Pinia 进行状态管理，数据持久化到本地存储：
- `src/store/checkin.ts` - 打卡数据存储

## TabBar 配置

底部导航栏包含4个页面：
1. 首页 - 统计和快捷入口
2. 足迹 - 地图页面
3. 记录 - 打卡列表
4. 我的 - 个人中心

## 使用说明

### 发布打卡
1. 点击首页的"发布打卡"按钮或地图页面的"打卡"按钮
2. 选择位置（自动获取当前位置或手动选择）
3. 添加图片（可选，最多9张）
4. 输入打卡内容
5. 点击"发布打卡"保存

### 查看地图
1. 在底部导航栏点击"足迹"
2. 地图上会显示所有打卡点
3. 点击标记点可查看详情
4. 点击"打卡"按钮可发布新打卡

### 查看记录
1. 在底部导航栏点击"记录"
2. 查看按日期分组的打卡记录
3. 点击记录可查看详情

## 注意事项

1. **地图功能**：需要在小程序后台配置地图相关权限
2. **位置获取**：首次使用需要授权位置权限
3. **图片上传**：当前版本图片存储在本地，如需上传到服务器，需要配置上传接口
4. **逆地理编码**：获取详细地址需要配置地图API key（腾讯地图/高德地图/百度地图）
